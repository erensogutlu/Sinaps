/* Sinaps | SMA Farkındalık ve Bağış Platformu */

Sinaps, SMA (Spinal Müsküler Atrofi) hastalığına dikkat çekmek ve tedavi süreçleri için bağış toplama simülasyonu sağlayan modern, interaktif ve kullanıcı deneyimi odaklı bir web projesidir. İsmini, sinir hücreleri arasındaki bağlantıyı sağlayan "sinaps" yapısından alır; bu da bağışçılar ile hastalar arasında kurulan hayati bağı simgeler.

* Özellikler : 

 -> Modern Glassmorphism Tasarımı
 -> Gece/Gündüz Modu
 -> Tam Mobil Uyumluluk (Responsive)
 -> Bağış Simülasyonu
 -> Canlı Bildirimler (Social Proof)
 -> Mağaza Vitrini

* Kullanılan Teknolojiler :

   -> HTML / CSS
   -> JavaScript

* Geliştirici : Eren Söğütlü

/* Synapse | SMA Awareness and Donation Platform */

Sinaps is a modern, interactive and user experience-oriented web project that draws attention to SMA (Spinal Muscular Atrophy) disease and provides a fundraising simulation for treatment processes. It takes its name from the "synapse" structure that provides the connection between nerve cells; This symbolizes the vital bond established between donors and patients.

* Features : 

 -> Modern Glassmorphism Design 
 -> Day/Night Mode 
 -> Full Mobile Compatibility (Responsive) 
 -> Donation Simulation 
 -> Live Notifications (Social Proof) 
 -> Store Showcase

* Technologies Used : 

 -> HTML/CSS 
 -> JavaScript

* Developer : Eren Söğütlü

⚠️ IMPORTANT NOTE : This project is a DEMO study developed for education and design purposes. No real money transfer or donation is made.

LINK : https://erensogutlu-sinaps.netlify.app/
